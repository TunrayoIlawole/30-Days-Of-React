What is props in a React component?
Props is a special keyword in React that stands for properties and is used to pass data from one component to another and mostly from parent component to child component.

How do you access props in a React component?
You access props by using the props object.

What data types can we pass as props to components?
Strings, Numbers, Arrays, Objects, Boolean, Function

What is a propTypes?
This is a package that helps us to assign the data types of the props we passed to the component.

What is a default propTypes?
This can be used when we want to have some default prop types for a component
