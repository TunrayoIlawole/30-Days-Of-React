What is the difference between a regular function and an arrow function?
Unlike regular functions, arrow functions do not own their own "this" and argument objects are not available in arrow functions.

What is a React component?
A react component is a small, reusable bit of code which is responsible for one part of the application's UI

How do you make a React functional component?
By creating a function through function declaration, function expression or arrow function and returning JSX

What is the difference between a pure JavaScript function and a functional component?
A functional component return JSX, while a pure JavaScript component does not return JSX.

How small is a React component?
The smallest a React component can be is a component that returns only a single HTML as JSX.
