2. What is state in React?
State is an object in React that allows React components to re-render whenever there is a change in state.

3. What is the difference between props and state in React?
Props are objects that can be used to pass values from parent components to child components, while states are objects that can be used to re-render the React component when their properties and values are changed.

4. How do you access state in a React component?
You access a state in a React component by using this.state

5. How do you set a state in a React component?
You set a state in a React Component by using this.setState
