2. What is inheritance and how do you make a child from a parent class?
Inheritance is a concept in object oriented programming that enables methods from a base class to be copied into a derived class.
To create a class inheritance, use the "extends" keyword

3. What is class based React component?
A class based component is a child of the React component an it has a built-in render method and may have a constructor.

4. What is the difference between functional React component and class based React component?
Functional React components are based on JavaScript functions, while class based components are based on JavaScript classes.

5. When do we need to use class based components instead of functional components?
When we need to change the state of the component, although new React versions have made it possible for that to happen using functional based components.

6. What are the use cases of class based components?
When you need to change the state of the React component.
