What is an HTML element?
An HTML element is a component of an HTML document that is defined by a starting tag and if it contains other contents, ends with a closing tag.

What is JSX?

What is babel?
Babel is a JavaScript transpiler that converts edge JavaScript into plain old ES5 JavaScript than can run in any browser.

What is a transpiler?
A transpiler is a tool that reads source code written in one programming language and produces the equivalent code in another language.
