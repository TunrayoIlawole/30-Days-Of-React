What is React?
React is a JavaScript library for building reusable user interface.

What is a library?
A library is a collected of precompiled routines that a program can use.

What is a single page application?
A single page application is an application that has only one html document.

What is a component?
Components are independent and reusable bits of code.

What is DOM?
DOM is the Document Object Model.

What is React Virtual DOM?
It is the representation of a DOM object that has the same properties as a real DOM object, but lacks the ability to directly change what is one the screen.
